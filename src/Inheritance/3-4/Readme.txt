Constructor Chaining: When we extend the class its super class constructor is called.
Method Overloading:Same name as that of method in the same class, but differing in number of parameters and type of parameters
Method Overriding:Same name as that of method in the subclass, number of parameters and type of parameters should not vary.
Hiding of Fields: public can be accessed anywhere, protected within subclass,private within the class.
Accessed Sublcasses:public and protected can be accessed.
